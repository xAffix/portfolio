Projects

Simple Port Scanner

This Python script allows you to scan for open ports on a target host within a specified port range. It uses the socket library to establish connections and identify open ports.

Security Audit

Conducted a thorough security audit for BotiumToys, a tech-forward children's toy company, to identify and mitigate vulnerabilities in digital and physical infrastructures. This audit focused on enhancing customer data safety and protecting the integrity of product designs and corporate information.

Simple URL Scanner

This was a very simple URL Scanner that used Google Safe Browsing API to check if the link was safe to open or if it was malicious. I used the help of Google Developer Docs and ChatGPT to build this project and learned with it as well.
